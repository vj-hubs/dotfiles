# Dotfiles

This repository contains dotfiles and configuration files for various tools and environments

## Repository Structure

| Directory | Description | Docs |
|-----------|-------------|------|
| `brew/` | Homebrew package manager configuration and Brewfile | [README](brew/README.md) |
| `cursor/` | Cursor IDE settings, keybindings, extensions, and tasks | [README](cursor/README.md) |
| `python/` | Python installation via Homebrew and pyenv on macOS | [README](python/README.md) |
| `scoop/` | Windows package manager (Scoop) configuration and scripts | [README](scoop/README.md) |
| `starship/` | Starship cross-shell prompt configuration | [README](starship/README.md) |
| `utils/` | Utility scripts | - |
| `vim/` | Vim configuration files | [README](vim/README.md) |
| `wsl/` | Windows Subsystem for Linux setup and configuration | [README](wsl/README.md) |
| `zsh/` | ZSH shell configuration and plugins | [README](zsh/README.md) |
